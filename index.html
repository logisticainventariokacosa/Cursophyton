 Cómo usar Python en tu Móvil

                    
No necesitas estar frente a tu PC para escribir y correr scripts de Python. Aquí tienes el manual paso a paso para configurar tu entorno móvil.

                


                

                    
                    
                


                

                    

                        1
                        
Descarga la aplicación

                        
Busca e instala **Pydroid 3 - IDE for Python 3** en Google Play Store. Es gratuita y la más estable para Android.

                    

                    

                        2
                        
Instala las Librerías (Montacargas)

                        
Abre Pydroid 3, ve al menú lateral (tres líneas) y selecciona **Pip**. En la pestaña "Quick install", puedes instalar **pandas** haciendo clic en su botón correspondiente, o escribir supabase o beautifulsoup4 en la pestaña "Library name" e instalar.

                    

                    

                        3
                        
Escribe y Corre tu Código

                        
Copia cualquier código del portal, pégalo en el editor de Pydroid 3 y haz clic en el botón amarillo de **Play** (esquina inferior derecha). Verás la consola ejecutando el script al instante en tu celular.

                    

                


                

                    

                        1
                        
Descarga la aplicación

                        
Descarga **Pythonista 3** (pago) o **Pyto - Python 3** (gratuito con funciones premium) en la App Store de iOS.

                    

                    

                        2
                        
Instalar Librerías en Pyto

                        
En Pyto, tienes una terminal integrada donde puedes escribir pip install pandas o pip install supabase directamente para agregar tus herramientas de inventario.

                    

                    

                        3
                        
Carga tus archivos locales

                        
Guarda tus reportes CSV en la aplicación "Archivos" de tu iPhone en la carpeta correspondiente a Pyto o Pythonista para que el código los pueda leer usando pd.read_csv("reporte.csv").
